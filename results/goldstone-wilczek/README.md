# Goldstone–Wilczek Results

This directory contains only the independently separable topological-current normalization evidence from the mixed legacy C6 commit. It establishes the upstream Paper 5 normalization (c_{GW}=1) within the recorded numerical tolerance, including cutoff, mass, and external-momentum checks.

Reproduce with:

```text
python -m scripts.goldstone_wilczek.c_gw_loop
```

Paper 3-owned C6, HS-vector, Walecka, vector-sign, and stabilization artifacts are excluded.

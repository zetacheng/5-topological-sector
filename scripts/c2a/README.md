# C2a Scripts

These scripts are migrated from `zetacheng/kappa-c2a` and retain the legacy algorithms. Transformations are limited to package-relative imports, the extraction of Euclidean gamma matrices from the mixed Fierz utility, and the aggregate script's default destination-data path.

Run from the repository root. The documented reproduction commands are in `results/c2a/README.md`. The box-only `skyrme_sign2.py` and `startup_regression.py` are lineage anchors, not the authoritative complete five-class evaluator.
